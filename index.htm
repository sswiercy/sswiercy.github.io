Game Maker

Spiele

Die folgenden Spiele und Programme wurden mit dem Game Maker von YoYo Games entwickelt und können hier heruntergeladen werden.

  • SWERVE (34,0 MB)
    Rennspiel aus der Vogelperspektive, dessen Strecken durch europäische Metropolen führen

  • öÖLFMETER (4,52 MB)
    Elfmeterschießen im Comic-Look, wahlweise gegen den Computer oder einen anderen Spieler

  • Coaster Rider (12,4 MB)
    Baue Strecken für eine Achterbahn, die in diesem physikbasierten Spiel zunächst alle Sterne durchfahren muss bevor sie ihr Ziel erreicht

  • Eurotower Minigame (4,03 MB)
    Umsetzung des klassischen "Hau den Lukas" im Kontext des Euro-Towers aus dem Europa-Park

  • Achterbahnfahrer (6,11 MB)
    Baue Achterbahnen und fahre sie aus der Egoperspektive mit der Möglichkeit die Geschwindigkeit zu steuern, um Bestzeiten zu fahren

  • Wasserwellen (1,03 MB)
    Wellensimulation mit verschiedenen Einstellungsmöglichkeiten und der Funktion, eigene Hintergrundbilder laden zu können

  • YATZEE (951 KB)
    Einfache Umsetzung des bekannten Würfelspiels mit Bestenlisten und bis zu drei Spalten, die frei genutzt werden können

  • LAN - Chat (965 KB)
    Chatprogramm für lokale Netzwerke

  • Block (635 KB)
    Rätselspiel, basierend auf dem Klassiker "Sokoban", mit insgesamt 50 Levels

  • Squarebattle (1,16 MB)
    Digitale Umsetzung des üblicherweise auf Papier gespielten Strategiespiels "Käsekästchen" mit wahlweise Computergegner oder Zweispielermodus

  • Pagode (837 KB)
    Auf dem Prinzip der "Türme von Hanoi" basierendes Rätselspiel

  • Pong! (842 KB)
    Version des Klassikers "Pong!", die nur über das Netzwerk gegen einen anderen Spieler gespielt werden kann

Erweiterungen

Die folgenden Erweiterungen für den Game Maker stellen Entwicklern zusätzliche Funktionalität zur Verfügung.

  • GM ScrRes DLL (35,7 KB)
    Ermöglicht das Auslesen aller unterstützten Grafikmodi des aktuellen Anzeigegeräts, bestehend aus Auflösung, Farbtiefe und Frequenz

  • OHS-System (281 KB)
    System aus Game Maker DLL und Serveranwendung, die es ermöglicht, einfache Online-Bestenlisten in Spiele einzubinden

  • GM RegEdit DLL (152 KB)
    Erweitert die Möglichkeiten, die Windows Registry aus dem Game Maker heraus manipulieren zu können

  • GM HTTP (429 KB)
    Stellt Funktionen für einen HTTP Client zur Verfügung und erlaubt asynchrone Requests

  • GM Audiere (537 KB)
    Stellt eine für den Game Maker kompatible API bereit, um die audio library "Audiere" nutzen zu können

  • Xor Crypt (1,35 MB)
    Ermöglicht das Packen und Verschlüsseln von Dateien, die aus dem Game Maker heraus geladen werden können, um beispielsweise Spieldateien zu schützen

JavaScript

Die folgenden Spiele wurden in JavaScript entwickelt und können daher direkt im Browser gespielt werden.

  • Autorennen
    Rennspiel mit insgesamt 10 Strecken, Tuning und einem Streckeneditor

  • Puzzle
    Klassisches Schiebepuzzle mit der Möglichkeit, eigene Motive zu laden

  • Billard
    Vereinfachte Billard Variante, in der menschliche oder computergesteuerte Spieler gegeneinander antreten

  • Maze Master
    Finde aus der Egoperspektive den Weg durch die insgesamt 10 Labyrinthe zum Ausgang

  • Snake
    Simple Umsetzung des Klassikers, bei dem man mit einer Schlange Elemente einsammeln muss, um zu wachsen und Punkte zu erhalten

  • Sudoku
    Programm zum Generieren und Lösen von Sudokus
