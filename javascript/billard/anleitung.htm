Anleitung
Spiel starten
Unter dem Billardtisch befinden sich zwei Buttons. Mit einem Klick auf "Einstellungen" können Sie ein Bereich öffnen und schließen, in dem sich die beiden Spieler einstellen lassen. Im ersten Auswahlmenü lässt sich einstellen ob der entsprechende Spieler von einem menschlichen Spieler oder vom Computer gesteuert werden soll. Je nach dem welche Einstellung gewählt wurde kann rechts daneben entweder der Spielername eingetragen oder die Schwierigkeitsstufe eines Computergegners eingestellt werden. Mit einem Klick auf "Neues Spiel" wird das Spiel mit den gewählten Einstellungen gestartet.


Spielregeln
Zu Anfang werden sieben rote und sieben blaue Kugeln und eine schwarze Kugel in einer Dreiecksform auf dem Tisch zufällig angeordnet. Es wird per Zufall bestimmt, wer anfangen darf.
Abwechselnd müssen beide Spieler die weiße Kugel möglichst so spielen, dass dadurch eine der farbigen Kugeln eingelocht wird, aber auf keinen Fall die Schwarze. Sobald zum ersten Mal eine farbige Kugel versenkt wird, wird dem zugehörigen Spieler die Farbe der eingelochten Kugel zugeordnet und dem Gegenspieler die jeweils Andere. Die Farben werden dann in der unteren Anzeige dargestellt. Von nun an müssen beide Spieler versuchen nur noch Kugeln ihrer Farbe einzulochen. Erst wenn alle Kugeln der Farbe eines Spielers versenkt wurden, darf dieser die schwarze Kugel einlochen. Wem dies als erstes gelingt, gewinnt. Wer jedoch die schwarze Kugel schon vorher einlocht, schenkt automatisch dem Gegner den Sieg.
Sollte während des Spiels die weiße Kugel eingelocht werden, ist unabhängig aller anderen eingelochten Kugeln der Gegenspieler am Zug, sofern nicht die schwarze Kugel das Spiel beendet. Wenn ein Spieler eine seiner zugehörigen Kugeln einlocht ist er nochmals an der Reihe.


Steuerung
Um die weiße Kugel zu spielen, muss mit der Maus auf den Spieltisch mit der linken Maustaste an die Stelle geklickt werden, wohin sich die Kugel bewegen sollen. Unter dem Billardtisch befindet sich eine Auswahloption um die Stärke des Stoßes einzustellen.
